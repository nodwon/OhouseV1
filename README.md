## 기능

### 게시판

- **주요 기능**
  - 게시글 CRUD
  - 댓글 CRUD
  - 조회순으로 정렬
  - 게시글 검색 기능
    - 제목, 본문, 닉네임

### 회원

- **주요 기능**
  - 회원 CRUD
  - 소셜 로그인 OAuth2 로그인 가능
    - Kakao, Naver, Google

### DevOps

- 사용 기술: AWS EC2, Nginx, AWS RDS
- **주요 기능**
  - 무중단 배포
